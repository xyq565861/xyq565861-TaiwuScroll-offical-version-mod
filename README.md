功能说明：

解除太吾求婚自身配偶数限制:解除配偶数量限制,说白了就是让你可以一夫多妻和一妻多夫（注意现在版本我只修改了对话触发的求爱，包括被动求爱，相亲大会，迷香阵之类的奇遇求爱事件没改，因为我没有那么后面的档没法测试）
解除对方其他关系限制：让你可以向师傅，结拜兄弟，还有其他部分血缘关系者求爱
解除求婚门派限制：忽视求婚对象的部分身分和门派限制（这会禁用远走高飞系列事件就是少林金刚五毒玄女派的私奔事件，因为会解除等级限制，而远走高飞会让npc脱离门派，我怕有后续bug）
太吾与目标年龄最低值：高于这个年龄就可以求爱，原版默认是16岁
保底告白成功率：对话触发的倾诉爱意成功率保底，忽视性别，年龄等因素，原版机制的话跟配偶数量，相貌，年龄差，魅力，性别取向都有关系，这里是保底率，正常成功率不会低于这个值。0表示没保底，3表示30%，8表示80%，以此类推
控制太吾孩子的性别：暂时在测试阶段，比较不稳定。对多胞胎，只能确定第一个孩子性别，往后每个孩子都有25%机率转变性别，和ea不同，正式版新生儿是生出来那一刻才生成的，ea版是怀孕那一刻，所以这个功能在出生那一刻才有效
负责的太吾父母:"原版机制下，非正常春宵一刻怀孕,或新生儿父母仅为情侣，而双方有其他正式配偶时，新生儿出生不会被赋予亲子关系，打开这个选项将让父母之一为太吾的婴儿，无视双方是否有其他配偶，都会添加亲子关系，仅开启时的婴儿出生事件有效"
debug：不要选，会在你的游戏文件夹生成一堆对你没用的日志
暂时只做了解除太吾配偶数量限制，身分与年龄要求的内容，后续的话计划原ea版太污绅士有的功能比如预测长相之类的都会有
但是这几天太不稳定了，尤其是我玩的档坏了2个了，因此测试开发其他功能有点麻烦，所以功能实现速度可能取决于我多久玩到游戏中后期
